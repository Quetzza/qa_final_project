## 📖 Descripción del Proyecto

Proyecto integral de testing para Urban Scooter, un servicio de alquiler de scooters eléctricos. Este proyecto demuestra mis habilidades en análisis de requisitos, diseño de casos de prueba, testing manual y automatizado, y reporte de defectos.

[Inicio](#-tabla-de-contenido)

## 🎯 Alcance de Pruebas

- Validar la funcionalidad completa de la aplicación Urban Scooter
- Asegurar la calidad en múltiples plataformas (Web, Móvil, API)
- Implementar metodologías de testing estructuradas
- Documentar defectos de manera profesional
